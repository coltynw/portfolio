Game of Life

I had a ton of fun making this one and is probably my favorite project. I have to stop myself from working on this because I get so caught up in adding additional options. I challenged myself to do the entire thing in base React without Hooks or State, which admittedly was a bad choice, as it made the code unnecessarily messy.

I learned how to deploy to Github pages with React. Usually I would just make a gh-pages branch and push to it, but that doesn't work if you use any dependancies. So I learned the difference between deploying and building. Deploy means to just copy the current artifacts to the server. Build means process all the code and compiles, generates additional code and packages it all to be delpoyed to the server.

Group Project

This application helps users track their macronutrients for optimal fitness. User authenication is fully supported, so you'll have to make an account and log in to use it, however, since we deployed the backend without paying Heroku, the database is refreshed once an hour. We collected the user data from the onboarding process and send to the backend. After submission we call new values to populate the dashboard elements. Along with highlighting the current day of the week and displaying sample images for the number of meals selected.

I worked with a team using React to make this one possible. With 2 other front end developers and 2 on backend, we had one week to piece it together and I'm very proud of what we managed to get done.
